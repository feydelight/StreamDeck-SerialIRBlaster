IR Commands

Write a list of commands to perform. Seperate the operations for your commands with a comma (',') and seperate different commands with a semi-colon (';'). The pattern is: [[Delay:int, Protocol: string, Address: int(HEX), Command: int(HEX);]*]
Example "500,NEC,0xF404,0x44;0,NEC,0xF404,0xE8;"
The above command will request to send the IR `BB44F404`, wait 500ms, then send the command `17E8F404`, and wait 0ms. Any invalid commands will be ignored, and written to log.
